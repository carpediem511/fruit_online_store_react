## Проект "Список покупок - интернет-магазин фруктов "

### ОСНОВНАЯ ИДЕЯ:

Демонстрация навыков разработки с использованием React, включая создание компонентов, работу с состоянием приложения и взаимодействие с пользовательским вводом.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

1**Список продуктов:
-каждый продукт представлен в виде карточки с изображением, названием, стоимостью за штуку и кнопкой "Добавить в корзину".
-при нажатии на кнопку "Добавить в корзину", цвет кнопки и текст на ней изменяются.

2**Корзина покупок:

-в корзине отображаются выбранные продукты с изображением, названием, количеством и стоимостью.
-пользователь может регулировать количество товара в корзине с помощью кнопок "стрелка вверх" и "стрелка вниз".
-для каждого продукта есть кнопка удаления из корзины.
-внизу корзины отображается общая стоимость заказа.

3**Модальное окно "Оформление заказа":

-при нажатии на кнопку "Оформить", появляется модальное окно с сообщением об успешном оформлении заказа.
-запрос на оформление заказа не реализован, но пользователь получает уведомление.

### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

- HTML, CSS, JavaScript;
- фреймворк: Create React App

Библиотеки:
-- @headlessui/react;
--classnames;
--react-scroll;
--tailwindcss;
--uuid4;


### ОСОБЕННОСТИ ПРОЕКТА:

-добавление товаров в корзину с изменением цвета и текста кнопки "Добавить в корзину".
-управление количеством товаров в корзине с использованием счетчика.
-возможность удаления товаров из корзины и расчет общей стоимости заказа.
-модальное окно заказа с обновлением страницы после успешного оформления.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Клонировать репозиторий с помощью git clone.
2. Установить зависимости с помощью yarn install.
3. Запустить проект с помощью yarn start.
Перейти по адресу http://localhost:1234 в браузере.
